# Bernese Mountain Dogs for Adoption 🐕

A professional, responsive website dedicated to Bernese Mountain Dogs available for adoption.

## Overview

This website showcases available Bernese Mountain Dogs for adoption and provides comprehensive information about the breed, adoption process, and health care guidelines.

## Features

✅ **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
✅ **Professional Styling** - Bernese Mountain Dog color scheme (black, rust, white)
✅ **Navigation** - Easy-to-use navigation with smooth scrolling
✅ **Sections Include:**
   - Hero Section with Call-to-Action
   - About the Breed
   - Breed Standards
   - Available Dogs for Adoption
   - Health & Care Guidelines
   - Adoption Process (5-step guide)
   - Contact Form
   - Footer

✅ **Interactive Elements** - Form validation, smooth scrolling, hover effects
✅ **Accessibility** - Semantic HTML structure

## Files

- **index.html** - Main HTML structure
- **styles.css** - Professional styling and responsive design
- **script.js** - Interactive functionality (form validation, smooth scrolling, animations)
- **README.md** - Documentation

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Customize the content (dog information, contact details, etc.)
4. Deploy to your web host or GitHub Pages

## Customization

### Update Dog Information
Edit the dog cards in the "Available Dogs" section of `index.html`:
```html
<div class="dog-card">
    <div class="dog-image">🐕</div>
    <h3>Dog Name</h3>
    <p><strong>Age:</strong> X years</p>
    <p><strong>Gender:</strong> Male/Female</p>
    <p>Description of the dog...</p>
    <button class="adopt-btn">Learn More</button>
</div>
```

### Update Contact Information
Edit the contact section with your organization's details:
- Email
- Phone
- Address
- Hours

### Modify Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #1a1a1a;
    --secondary-color: #8B4513;
    --accent-color: #FFFFFF;
    --light-bg: #f5f5f5;
    --text-dark: #333;
    --text-light: #666;
}
```

## Color Scheme

- **Primary Black**: #1a1a1a
- **Rust Brown** (Secondary): #8B4513
- **White** (Accent): #FFFFFF
- **Light Gray** (Background): #f5f5f5

## Deployment Options

### GitHub Pages
1. Push this repository to GitHub
2. Go to Settings → Pages
3. Select main branch as source
4. Your site will be live at `https://yourusername.github.io/repo-name`

### Other Hosting
- Upload files to any web hosting service
- Ensure index.html is in the root directory
